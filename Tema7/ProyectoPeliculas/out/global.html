manejarScroll()

Esto se encarga de manejar el exceso de peticiones ajax. Cada vez que se hace scroll y se llega al límite inferior, llamará al método loadDoc, el cual hará peticiones ajax y llamará a la vista para que maquete los resultados, sin embargo, para evitar que se sobrecargue de peticiones ajax al hacer exceso de scroll, usará una variable "comprobar", que siendo false impedirá nuevas peticiones, y se volverá true tras terminar la petición que realiza loadDoc.
