What is a cookie?

Son simples archivos de texto que almacenan los servidores web en las correspondientes máquines del usuario. Los archivos contienen información en un par nombre-v entre los múltiplos usuarios que acceden al sistema. Estas cookies serán accedidas por el sitio web que las ha generado cada vez que el usuario acceda a la web.


What cookies are used for?

  • Almacenar la información de estado actual de un sitio web en la computadora del usuario
  • Sirven para controlar el número de usuarios que acceden a la página web, y la cantidad de veces que lo hace cada uno
  • Recordar preferencias y configuraciones, para de esta forma hacer la página algo más personal de cara al usuario
  • Para rastrear el tiempo que un usuario particular pasa en el sitio web y para encontrar los enlaces en los que el usuario hace clic en ese sitio web en particular.

How Cookies Are Initialized in Browsers?

Cuando escribimos una URL, el navegador envía una solicitud al sitio web para la página. Al hacer esto, el navegador también busca una cookie en la computadora del usuario. Si el navegador encuentra una cookie para ese sitio web en particular, enviará ese par de nombre-valor a ese servidor web asociado con la URL. El servidor recibe el par nombre-valor y la solicitud de la página y utiliza los datos de la cookie para producir la experiencia de usuario única al proporcionar preferencias particulares y esquemas de color.


What types of Cookies do you know?

  • Cookies transitorias

    • Este tipo de cookie se almacena en la memoria RAM y contiene una ID de sesión que almacena información sobre la sesión de navegación del usuario. Se eliminan de la computadora tan pronto como el navegador se apaga.

  • Cookies persistentes

    • Este tipo de cookies se almacenan en el disco duro de la computadora y contienen información sobre las preferencias del usuario para un sitio web. Estas preferencias se pueden usar para sesiones de exploración adicionales y se conservan mientras el usuario las permita. Se almacenan permanentemente y no se eliminan cuando se cierra el navegador.

  • Flash cookies

    • son pequeños archivos Flash, como videoclips o gifs almacenados en la computadora de un usuario por sitios web. No se pueden eliminar mediante ningún mecanismo desde el nivel del navegador. Solo se pueden eliminar manualmente o mediante algunos complementos especiales.


What are the common risks of cookies?

  • Ataque de falsificación de solicitud entre sitios (XSRF)
  • Fijación de sesión
  • Creación de scripts entre sitios
  • Ataque de lanzamiento de cookies
  • Ataque de desbordamiento de cookies
